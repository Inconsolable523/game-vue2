# vueMemory

#说明
>使用vue-cli自动搭建的一个记忆小游戏，很清晰的一个vue2+vuex的项目。

>移动端和pc端任意畅玩 😁


>很适合新手平静心情 😊或者学习vue。
#项目运行

``` bash
# 克隆到本地

# 安装依赖
yarn install

# 开启本地服务器localhost:8088
npm run dev

# 发布环境
npm run build
```
